# vuejs-template-sama

## Instala as dependências
```
npm install
```

### Compila e seta refresh automático para desenvolvimento
```
npm run serve
```

### Builda e deixa tudo menor para por em produção 
```
npm run build
```

### Encontra problemas no projeto
```
npm run lint
```
